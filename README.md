# data-modeling
Este projecto contiene un diagrama de modelamiento de datos que muestra claramente cómo se relacionan los datos públicos entre sí para estudiar el mercado de hidrocarburos en Colombia.


- Producción Fiscalizada de Crudo Consolidada ([https://www.datos.gov.co/Minas-y-Energ-a/Produccion-Fiscalizada-Crudo-Consolidada/fdvb-hsrf/about_data]) - (ANH)

- Producción Fiscalizada de Gas Consolidada ([https://www.datos.gov.co/Minas-y-Energ-a/Produccion-Fiscalizada-Gas-Consolidada/5dux-bfvx/about_data]) - (ANH)

- Consolidación de Liquidación de Regalías por Campo ([https://www.datos.gov.co/Minas-y-Energ-a/Consolidaci-n-de-liquidaci-n-de-regal-as-por-campo/j7js-yk74/about_data]) - (ANH)

- Impuesto de Transporte ([https://www.datos.gov.co/Minas-y-Energ-a/Impuesto-de-Transporte-MinEnerg-a/7s8e-7zcb]) - (Ministerio de Minas y Energía)

- DIVIPOLA - Código Municipios ([https://www.datos.gov.co/Mapas-Nacionales/DIVIPOLA-C-digos-municipios/gdxc-w37w/about_data]) - (DANE)
